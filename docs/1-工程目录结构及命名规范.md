## 2. 文件命名规范

- **目录**：
  使用小写字母和连字符（kebab-case），如 `components/`、`user-management/`
- **Vue 组件**：
  使用 PascalCase，如 `UserProfile.vue`、`DataTable.vue`
- **TypeScript/JavaScript**：
  使用 camelCase，如 `userService.ts`、`formatDate.ts`，工具函数使用 `utils.ts`、`helpers.ts`，类型定义使用 `types.ts`、`interfaces.ts`
- **样式文件**：
  使用 kebab-case，如 `user-profile.scss`、`button.scss`，全局样式使用 `index.scss`、`variables.scss`
- **测试文件**：
  与源文件同名，添加 `.test` 或 `.spec` 后缀，如 `userService.test.ts`、`UserProfile.spec.ts`
